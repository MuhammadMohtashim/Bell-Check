# Contributing to Bell Detector

First off, thank you for wanting to help! To keep the project's legal status clear and allow for future commercialization by the original author, we require all contributors to agree to our CLA.

### 📜 Contributor License Agreement (CLA)
By submitting a Pull Request, you grant the project owner a non-exclusive, irrevocable, worldwide, royalty-free license to use, modify, and commercialize your contribution as part of this project. You retain the copyright to your specific changes, but you allow us to include them in any future versions (including commercial ones).

### How to contribute
1. Fork the repo.
2. Create a feature branch.
3. Sign the CLA via the automated comment on your Pull Request.
4. Keep your code clean and follow the existing indentation style!
